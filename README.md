## 说明：

本安装程序是将小狼毫输入法的原安装程序以及雾凇拼音配置打包在一起的集成安装包。

雾凇拼音是小狼毫输入法的一套完整配置，包含输入方案（全拼、常见双拼）、长期维护的开源词库及各项扩展功能。

该简化安装包对雾凇拼音做了些许的调整， 同时增加了清华双拼输入方案。
安装时可选是否安装这些修改。主要修改如下：

1. 候选项改为九个。
2. 切换拼音方案只保留 Ctrl 加波浪号一个快捷键。
3. 解决数字后的标点不自动上屏问题。
4. 可选默认英语输入模式。
5. 增加清华双拼方案。
6. 清华双拼方案支持联想码定字功能（由 Chen.Chilin 老师整理）。
7. 安装时即可选择您习惯的拼音方案。当然之后在使用过程中，可以随时使用 Ctrl 加波浪号切换其他拼音方案。
8. 右侧 Shift 也作为中英切换。
